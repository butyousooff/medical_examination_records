# Учёт результатов диспансеризации
Консольное приложение для управления медицинскими данными воспитанников детского сада. Загружает, редактирует и сохраняет информацию о результатах осмотров специалистов в текстовом файле.
Возможности:
  Загрузка/сохранение данных в файл database.txt
  Добавление, редактирование, удаление записей о воспитанниках
  Три вида медицинских отчетов с сортировкой методом выбора
  Автоматическое подтверждение сохранения перед выходом
  Готовые демо-данные (25 записей воспитанников с различными заключениями)
  Полная валидация вводимых данных с обработкой ошибок
Отчеты:
  1. Полный список всех детей (количество здоровых заключений ↓, фамилия ↑)
  2. Список детей заданной группы (год рождения ↑, месяц ↑, день ↑)
  3. Список детей, нуждающихся в лечении (группа ↑, фамилия ↑)
Запуск python main.py Файл database.txt создается автоматически. При первом запуске можно создать демо-данные или добавить записи вручную.

Требования: Python версии 3.13 и выше
